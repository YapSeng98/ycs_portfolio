SAP Service Helpdesk

A structured IT support system with three dedicated modules — Authorization, Incident, and Change Request — enabling users to raise, track, and resolve SAP-related issues through a streamlined approval and escalation workflow.

  • Three modules: Authorization requests, Incident reporting, and Change Request management
  • User-friendly ticket submission form with issue categorisation, description, and priority selection
  • Automated ticket routing to the appropriate support team based on request type and module
  • Multi-level approval workflow for authorization and change requests with configurable approvers
  • Real-time ticket status tracking for requestors from submission through resolution
  • Automated email notifications and reminders via Power Automate at each workflow stage
  • Support team dashboard to monitor open, in-progress, and resolved tickets
  • All ticket data and records stored securely in SharePoint lists

TP Document Management System

A centralised document management system built on Power Apps and SharePoint, enabling structured storage of company files, project documents, vendor records, and contracts — with automated PDF renaming to enforce standardised naming conventions across the organisation.

  • Centralised repository for company files, projects, vendor records, and contract documents
  • Automated PDF renaming engine — renames uploaded files according to predefined format rules
  • Dynamic folder structure organised by company, project type, and document category
  • Upload documents directly from Power Apps with automatic metadata tagging
  • Search and retrieve documents by company, project, vendor, or contract reference
  • SharePoint integration for secure, permission-controlled document storage
  • Eliminates manual filing errors through enforced naming and categorisation rules
  • User-friendly interface designed for non-technical staff to manage documents with ease
